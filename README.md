# BT Bingo!

To install: <br />
git clone https://github.com/Danny-Robinson/bt-bingo.git <br />
cd bt-bingo <br />
npm install <br />
Install <a href="https://www.mongodb.com/download-center#community">mongo</a> <br />
DEV - npm run dev | PROD - npm start <br />
